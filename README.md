# AI-ENABLED-MILITARY-MEDICAL-PRIORITIZATION-SYSTEM-FOR-CRITICAL-BATTLEFIELD-CARE

An AI-driven system designed to assist in prioritizing injured soldiers and supporting medics with real-time medical decisions in critical environments.

---

## Overview

In high-pressure situations such as battlefields, timely and accurate decisions play a crucial role in saving lives. Traditional triage methods rely heavily on manual judgment, which can become slow and inconsistent when dealing with multiple casualties.

This project introduces a machine learning-based approach to improve the efficiency of medical response. By analyzing patient vitals and conditions, the system classifies the severity level and provides basic first-aid guidance. The aim is to support medics in making faster, more informed decisions during emergency scenarios.

---

## Project Status

⚠️ This project is currently under development and is in the prototype stage.

While the core functionalities such as severity prediction and basic decision support are implemented, the system is not yet production-ready. Further improvements are being made to enhance accuracy, scalability, and real-time integration.

---

## Key Features

-  **Severity Prediction** using machine learning models  
-  **First-Aid Suggestions** based on patient condition  
-  **Multi-Patient Handling** for better prioritization  
-  **Alert System** for identifying critical cases  

---

## Tech Stack

- Python  
- Scikit-learn  
- TensorFlow / PyTorch  
- Pandas, NumPy  
- Flask / Streamlit  

---

## Objective

The primary objective of this project is to assist in faster and more reliable medical decision-making during emergency situations. By reducing delays and supporting prioritization, the system aims to improve response efficiency, especially during the critical “golden hour.”
